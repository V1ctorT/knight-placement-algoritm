# Knight Placement Algorithm

This project implements a search-based algorithm in Python to solve a variation of the knight placement problem on an NxN chessboard.

## Features
- Depth-First Search (DFS) and backtracking
- Constraint-based placement using knight movement rules
- Generation of multiple valid solutions
- Export of solutions to a text file

- ##How to run
- 1. Run the script: python knights.py
  2. Enter board size N
  3. Solutions will be saved in soluciones.txt
